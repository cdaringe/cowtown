# security

how to build a secure application cannot reasonably be covered in guide of
recipes. however, there is a moderate selection of off-the-shelf tooling at
your disposal which you may consider installing into your application.

- [common sense security headers - helmet](https://github.com/venables/koa-helmet)
- [cors](https://github.com/koajs/cors)
- [rate limiting - ratelimit](https://github.com/koajs/ratelimit)
- [koa-toobusy](https://github.com/nswbmw/koa-toobusy)

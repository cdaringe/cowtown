# create a hello world server

let's register a single middleware.  the middleware considers nothing about the
details of your request--it blindly sets `'hello world'` to the body.

<div><Demo /></div>

// hello-world/server.ts
import Koa from 'koa'

  app.listen(3000)
}
start()
